# Archived Compose Files

This folder stores deprecated Docker Compose files preserved for historical reference.

Retired from repository root:

- `docker-compose-updated.yml`
- `docker-compose.prod.yml`
- `docker-compose.portainer.yml`

Use canonical files at repository root instead:

- `docker-compose.yml`
- `docker-compose.dev.yml`
- `docker-compose.dev.portainer.yml`
- `docker-compose.prod.portainer.yml`

Validation helpers:

- `scripts/validate-compose.ps1`
- `scripts/validate-compose.sh`
